                         Squarescape

  What is it?
  -----------
  This is a simple game made in Javascript. It is controlled by the use of One Button. 
  The goal of the game is to get as high of a score as you can before you die.
  Score is gained by jumping over gaps without falling to your death.
  This game is made for a project at school.

                         Sweets Stampede

  What is it?
  -----------
  This is a simple game made in AS3. It is controlled by the use of One Button.

  Score is gained by collecting the sweets that fall from the game ceiling.
